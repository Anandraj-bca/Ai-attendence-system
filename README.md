# AI Smart Attendance System

This project is an AI-based attendance system that uses face recognition to automatically mark attendance.

## Features
- Real-time face detection
- Face recognition using encoding
- Automatic attendance logging with timestamp
- Prevents duplicate entries

## Technologies Used
- Python
- OpenCV
- face_recognition
- NumPy

## How It Works
The system captures video input, detects faces, and compares them with stored images. If a match is found, attendance is marked automatically.
